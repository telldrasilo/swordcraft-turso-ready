## Обзор проекта

**SwordCraft: Idle Forge** — браузерная инкрементальная игра в сеттинге средневековой кузницы.

| Параметр | Значение |
|----------|----------|
| Жанр | Incremental / Idle / Tycoon |
| Платформа | Браузер (Desktop + Mobile) |
| Стек | Next.js 16, React 19, Zustand, Tailwind CSS 4, shadcn/ui |
| Строк кода | ~14 500 |
| Компонентов | 85+ файлов |

## Реализованные системы

### ✅ Система ресурсов
- 6 основных + 5 редких + 6 переработанных ресурсов
- Автогенерация каждые 100мс
- Ofлайн-прогресс до 8 часов

### ✅ Система рабочих
- 8 классов рабочих (ученик → чародей)
- 5 характеристик (скорость, качество, выносливость, интеллект, лояльность)
- Механика стамины и отдыха
- Прокачка до 50 уровня

### ✅ Система крафта
- 16 рецептов оружия (5 тиров)
- 6 градаций качества (плохое → легендарное)
- Система износа и ремонта (5 типов ремонта)
- 20 уровней мастерства кузнеца

### ✅ Система гильдии
- **Система модификаторов v5** — расширяемая архитектура
- **7 провайдеров модификаторов:**
  - Боевые характеристики (Power/Precision/Endurance/Luck)
  - Уровень и редкость
  - Черты характера (12 тегов)
  - Мотивации (8 типов)
  - Социальные теги (8 типов)
  - Сильные/слабые стороны (20 шт)
  - Стиль боя (10 стилей)
- **База известных искателей:**
  - Лимит 25 искателей
  - Повторное появление 30-60%
  - Требуется 3 миссии для контракта

### ✅ Система экспедиций
- 5 типов миссий (охота, разведка, зачистка, доставка, магическая)
- 5 уровней сложности
- Симуляция поиска искателей (30-60 сек)
- Детальный расчёт успеха/золота/душ/износа

### ✅ Система зачарований
- Жертвоприношение оружия → эссенция душ
- 6 школ зачарований (огонь, лёд, жизнь, защита, сила, скорость)
- Максимум 2 зачарования на оружии

## Связи характеристик с механиками

| Характеристика | Влияние |
|----------------|---------|
| Power | Души войны: +(power-25)×0.5% |
| Precision | Шанс успеха: +(precision-25)×0.3% |
| Endurance | Износ оружия: -(endurance-25)×0.4% |
| Luck | Шанс крита: 5+(luck-25)×0.2% |
| Уровень | Бонус к успеху и душам |
| Черты характера | Шанс согласия + эффекты экспедиции |
| Мотивации | Бонусы к наградам |
| Соц. теги | Модификатор золота |
| Стиль боя | Бонусы к типам миссий |

## Ключевые принципы разработки

### Zustand селекторы
```tsx
// ✅ Правильно
const gold = useGameStore((state) => state.resources.gold)

// ❌ Запрещено (infinite loop)
const { gold, iron } = useGameStore((state) => ({...}), shallow)
```

### Мемоизация
- `useMemo` — для сортировки/фильтрации массивов
- `React.memo` — для компонентов в списках
- `key={item.id}` — стабильные ключи, не index

### Структура файлов
- Компоненты по фичам (`components/guild/`, `components/forge/`)
- Slices для state-логики (`store/slices/`)
- Провайдеры для модификаторов (`lib/modifier-system/providers/`)
