# Система крафта v2.0

> **Статус:** Утверждённая концепция
> **Дата:** 2026-03-22

---

## 1. Философия

| Принцип | Суть |
|---------|------|
| Рецепт = Схема | Определяет части и форму, не материалы |
| Свобода материалов | Игрок выбирает материалы для каждой части |
| Наблюдаемый процесс | Автоматический крафт с атмосферными сообщениями |
| Мастерство = Качество | Качество зависит от навыка кузнеца |
| Масштабируемость | Легко добавлять новые материалы, техники, рецепты |

---

## 2. Основной цикл игры

```
КРАФТ: Рецепт + Сырьё + Техники → Обработка → Детали → Оружие
   ↓
ПРИКЛЮЧЕНИЯ: Оружие + Искатель → Экспедиция → Душа Войны + Износ
   ↓
АЛТАРЬ: Оружие с Душой → Жертвоприношение → Эссенция Души
   ↓
ЗАЧАРОВАНИЯ: Эссенция + Оружие → Усиление → Более мощное оружие
```

## 7. Проверка инвентаря 🔄

### 7.1 Требования

**Перед стартом крафта:**
1. Проверить наличие всех материалов
2. Показать недостающие
3. Заблокировать старт при нехватке

### 7.2 Структура

```typescript
interface CraftRequirement {
  materialId: string
  quantity: number
  available: number    // из инвентаря
  sufficient: boolean
}

interface CraftRequirementsCheck {
  canCraft: boolean
  requirements: CraftRequirement[]
  missing: CraftRequirement[]
  totalCost: {
    materials: CraftRequirement[]
    fuel?: { id: string, quantity: number }
  }
}
```

### 7.3 Расчёт для сплавов

Сталь требует: `железо × 3 + уголь × 1` на 1 ед. стали.

```typescript
// Если выбрана сталь для лезвия (2 ед.):
// Нужно: железо × 6, уголь × 2

function calculateRawMaterials(material: Material, quantity: number) {
  if (material.recipe) {
    // Это сплав — раскрываем рецепт
    return material.recipe.inputs.map(input => ({
      materialId: input.resourceId,
      quantity: input.quantity * quantity
    }))
  }
  // Сырьё — прямая запись
  return [{ materialId: material.id, quantity }]
}
```

## 9. Погружение (фразы и мини-игры) ⏳

### 9.1 Принцип

- Интерактив = маленькие бонусы (без штрафов за пропуск)
- Фразы = технические детали, без пафоса
- 1-2 мини-игры за крафт

### 9.2 Мини-игры

| Мини-игра | Этап | Бонус |
|-----------|------|-------|
| **Температура закалки** | `fin_hardening` | +1-3% качества |
| **Ритм ковки** | `form_forging` | +1-2% качества |

### 9.3 Пассивные клики

| Объект | Эффект |
|--------|--------|
| Искры при ковке | +5 опыта кузнеца |
| Звук молота | +1% скорости этапа |
